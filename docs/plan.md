만드는 것
유튜브 요리 영상을 검색하면 재료와 레시피를 자동으로 뽑아주는 웹서비스. 재료 클릭 시 쿠팡 구매 링크로 연결.
사용 기술 (2026년3월15일 기준 최신 스택으로 진행)
Next.js 풀스택 (App Router + API Route), Supabase, GPT-4o mini, YouTube Data API, 쿠팡 파트너스 API, Tailwind CSS
핵심 흐름 1. 검색어 입력 → YouTube API로 영상 조회 → 요리 채널만 필터링해서 리스트 표시 2. 영상 클릭 → 자막 가져와서 GPT로 재료/레시피 분석 → 모달로 표시 3. 같은 영상 재검색 시 → Supabase에 저장된 분석 결과 재사용 (API 비용 절감) 4. 재료 클릭 → 쿠팡 파트너스 링크로 이동
주요 결정 사항
∙ 채널 필터링: 구독자/조회수 기준 + 화이트리스트 병행
∙ 자막: 자동 생성 자막 포함, 한국어만 허용
∙ 긴 자막: 토큰 단위로 분할 분석 후 병합
∙ 검색 기록: localStorage에 탭 형태로 저장
∙ 쿠팡 API 키 발급 했다는 가정하에 진행
